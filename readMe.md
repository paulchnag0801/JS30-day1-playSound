# **project01 - JavaScript playSound**

![](https://github.com/paulchnag0801/JS30-day1-playSound/blob/main/js-playSound-cover.png)


## **主題**
透過JS使鍵盤按下(或滑鼠點擊)後播放出對應按鍵的聲音，並同時產生一個特效，  
在按下其他鍵(或滑鼠點擊)後會關閉該特效並於新按鍵中啟用。
